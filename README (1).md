# Simulação MCMV — Pedro Pires Imóveis

Gerador de apresentação de simulação do Minha Casa Minha Vida para clientes do Parque Ilhabela.

## Como usar

1. Acesse a página principal
2. Digite o nome do cliente
3. Cole o Ctrl+A do simulador da Caixa Econômica Federal
4. Clique em **Gerar apresentação**
5. A apresentação abre numa nova aba — limpa, sem dados técnicos que assustam

## Deploy no Vercel

1. Faça upload deste repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e importe o repositório
3. Deploy automático — sem configuração necessária

## Arquivos

- `index.html` — Página de entrada (cola o texto aqui)
- `apresentacao.html` — Apresentação renderizada para o cliente

## Dados extraídos automaticamente

- Valor do imóvel
- Parcela mensal
- Entrada necessária + desconto
- Composição da parcela (amortização, seguros, taxa admin)
- Renda familiar, FGTS, cidade, prazo de obra
